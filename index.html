Введение во Frontend разработку

Frontend разработкаотвечает за созданиеинтерфейся пользовольской части веб-сайта или веб-приложения. Она включает в себя работу с HTML , CSS и JavaScript.

Языки Frontend разработки

  • "HTML (Hyper Text Markur Language)- используется для структурирования содержимого веб-страницы"
  • "CSS (Cascading Style Sheets) - используется длястилизации веб-страницы и создания её внешнего вида"
  • "JavaScript - используется для добавления интерактивностина веб-страницу, такой как анимации, обработка событий и динамическое обновлениесодержимого"

Инструменты Frontend разработки

  1. Visual StudioCode - мощный редактор кода с поддержкой множества языков и плагинов
  2. React - библтотека для создания пользовательских интерфейсов
  3. Sass - препроцессор CSS с расширинными возможностями
  4. Wabpack = сборщик модулей для JavaScript приложений
